Web App realizada por Daniel Bermúdez


Mediante la selección de Comunidad y de Ciudad, se muestra el tiempo previsto para el día actual y los 4 siguientes.

Adicionalmente, se muestra la previsión de las temperaturas, inclueyndo mínimas y máximas,en un gráfico.

Se puede acceder al código fuente en el repositorio GitHub.



Adicionalmente, pueden acceder a una app web WikiCountry, así como a un juego Memory Cards.
